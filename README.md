# [fresh] ~/.dotfiles

## Installation

``` sh
FRESH_LOCAL_SOURCE=twe4ked/dotfiles bash -c "`curl -sL get.freshshell.com`"
chsh -s /bin/zsh
```

add it to your shell (e.g. ~/.zshrc)
``` sh
source ~/.fresh/build/shell.sh
```

My dotfiles are managed by [fresh].

[fresh]: https://github.com/freshshell/fresh
